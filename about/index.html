关于我

本人主要从事 C++ 后端基础设施、运行时与并发库、游戏服务端开发框架和解决方案、可观测性、构建工具链和跨平台工程也折腾了不少。

这个博客的目的是记录工程里反复出现的问题、踩坑记录和设计复盘，并沉淀成可复用的底层能力。包括但不限也方便有兴趣的小伙伴们一起交流探讨。

开源贡献

我主要维护或长期参与的开源项目：

  • OpenTelemetry C++（CNCF 项目）：核心贡献者 / Approver，2021 年至今。
  • libcopp ：跨平台 C++ 协程库，已收录 vcpkg。
  • xresloader ：一套跨平台的 Excel 转表工具链（文档站），核心引擎可把 Excel 导出为 Protobuf / Msgpack / Lua / JS / JSON / Xml / UE DataTable。其它组件：
    • xresconv-cli — 跨平台批量转表 CLI
    • xresconv-gui — 跨平台批量转表 GUI
    • xres-code-generator — 读表代码生成器
    • xresloader-dump-bin — 二进制 dump 工具
  • atframework：高性能、全异步、跨平台、去中心化的游戏服务器框架生态，包含 Utility 库、libatbus 高性能进程间通信\(小包单线程百万级QPS，大包单线程12Gbps/5Gbps吞吐\)、libatapp 应用框架、atsf4g-co 完整解决方案、cmake-toolset 构建系统、atgateway 的 C# 适配（可用于 Unity） 等等。
  • AICodeReviewer（AICR）：自托管的多 VCS AI 代码审查编排服务。把 GitHub / Gitea / GitLab / P4 / SVN 等触发统一成一条 review 流水线，在沙箱里跑自选 agent（Kilo Code、Claude Code、opencode、Zoo Code、Copilot CLI），产出结构化结论回写到 PR 评论、issue 或 IM 机器人；带密钥脱敏、Prometheus 指标和 LLM 成本看板。文档站。
  • hiredis-happ：Redis 高可用接入，较早支持 redis cluster，带自动重试、断线重连、按需连接。
  • upload-to-github-release：GitHub Action，已上架 Marketplace。

其它还有 ddns-cli、wxwork_robotd（企业微信机器人）、coredns-filter / coredns-nftables、WP-Code-Highlight.js，以及这个博客的主题 hugo-theme-distinctionpp。生命不息，折腾不止，平常写的小玩具就不一一列举了。
